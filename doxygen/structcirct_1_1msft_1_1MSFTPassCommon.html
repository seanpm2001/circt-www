)
protected

Update all the instantiations of 'mod' to match the port list.

For any output ports which survived, automatically map the result according to newToOldResultMap. Calls 'getOperandsFunc' with the new instance op, the old instance op, and expects the operand vector to return filled. getOperandsFunc can (and often does) modify other operations. The update call deletes the original instance op, so all references are invalidated after this call.
