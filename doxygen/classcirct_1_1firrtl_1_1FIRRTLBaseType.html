◆ getMaxFieldID()

uint64_t FIRRTLBaseType::getMaxFieldID ( )

Get the maximum field ID of this type.

For integers and other ground types, there are no subfields and the maximum field ID is 0. For bundle types and vector types, each field is assigned a field ID in a depth-first walk order. This function is used to calculate field IDs when this type is nested under another type.
